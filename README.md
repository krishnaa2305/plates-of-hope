### Colors
- **Primary Green**: `#16A34A`
- **Dark Green**: `#19331A`
- **Light Green**: `#4ADE80`
- **Gradient**: Linear gradient from `#16A34A` to `#059669`

### Fonts
- **Inter**: Primary font for body text
- **Montserrat**: Used on donation and request pages
- **Poppins**: Used for hero titles (800 weight)
- **Quicksand**: Used on login page
- **Font Awesome**: For icons

### Pages

1. **Home Page** (`index.html`)
   - Hero section with CTA buttons
   - Impact statistics
   - How it works section
   - Testimonials
   - Call-to-action section

2. **Login Page** (`login.html`)
   - Two-column layout
   - Left: Hero image with branding
   - Right: Login form with social login options
   - Password toggle functionality

3. **Donate Page** (`donate.html`)
   - Hero section with statistics
   - Partner locations
   - Donation form

4. **Request Page** (`request.html`)
   - Food request form
   - Security badges

5. **About Page** (`about.html`)
   - Mission section
   - What we do section
   - Our story section
   - Values section
   - Join our mission CTA
